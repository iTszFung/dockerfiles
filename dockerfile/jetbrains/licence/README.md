## TL;DR;

## 为何使用本容器?

一个基于 Alpine 最小的 **Docker 容器** 镜像

**NOTE:** 所有镜像基于 Docker 17.01 或更新版本.

## 获取镜像

推荐从 [JoyTech Private Registry](joytech) 拉取预构建版本

```bash
$ docker pull joytech/:latest
```

## 构建镜像

构建自己的版本，可根据脚本修改自己想要的属性

```bash
$ ./build
```
